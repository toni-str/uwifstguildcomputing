## Package Installs to have/ how to run locally:

1. ##### Download react: https://nodejs.org/en/download
2. ##### Install these packages next:
3. ##### npm install lucide-react
4. ##### npm install @radix-ui/react-toast
5. ##### Open terminal or CMD and navigate to the saved un-zipped file(Computing-ROADMAP-WEBSITE) then execute the following command one after the other.
6. ##### cd app
7. ##### npm run dev
